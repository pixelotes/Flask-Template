Flask App Template

A secure and scalable starting point for your Flask applications.

Features Included:

  • Authentication: User login, logout, and password management.
  • OAuth: Integrated with Google OAuth via Flask-Dance.
  • Security:
    • CSRF Protection (Flask-WTF)
    • Rate Limiting (Flask-Limiter)
    • Anti-Brute Force
    • New IP Detection & MFA Email
  • Database: SQLAlchemy ORM with User model.
  • UI: Bootstrap 5 with responsive sidebar and base templates.

How to Use:

1. CSRF Protection

CSRF protection is enabled globally. Include the hidden token in your forms:

<input type="hidden" name="csrf_token" value="{{ csrf_token() }}">
2. Flash Messages

Send messages from your routes using `flash()`:

flash('Operation successful!', 'success')
flash('Something went wrong.', 'danger')

They are automatically rendered in `base.html`.

3. Creating New Routes

Create a new Blueprint in `src/routes/` and register it in `src/__init__.py`.
